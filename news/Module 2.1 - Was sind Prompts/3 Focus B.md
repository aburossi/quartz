# Experimentieren mit Prompts
### Auftrag Intermediate Level A
#### Prompt-Dimensionen im Einsatz

> [!abstract] Prompt-Dimensionen im Einsatz
> In diesem Auftrag lernen Sie, wie man Prompts mit verschiedenen Dimensionen (Direktheit, Spezifität, Tonalität) erstellt, um die Ausgabe eines Sprachmodells gezielter zu beeinflussen. Sie experimentieren mit ChatGPT und analysieren die Auswirkungen der verschiedenen Prompt-Dimensionen.

>[!todo]- Ablauf
>- Bilden Sie Gruppen und wählen Sie ein Thema aus der Themenliste.
>- Formulieren Sie für das Thema drei verschiedene Prompts, die jeweils eine andere Dimension (Direktheit, Spezifität, Tonalität) in den Vordergrund stellen.
>- Geben Sie Ihre Prompts in ChatGPT ein und beobachten Sie die generierten Texte.
>- Vergleichen Sie die generierten Texte und analysieren Sie, wie sich die Prompt-Dimensionen auf die Ausgabe von ChatGPT auswirken. 
>   - Welche Dimensionen haben den grössten Einfluss auf die Qualität und den Inhalt der Texte?
>- Tragen Sie Ihre Prompts, die generierten Texte und Ihre Beobachtungen in die Vergleichstabelle ein.

>[!quote]- Prompting
> Bei der **Prompt-Erstellung** können Sie verschiedene **Dimensionen** nutzen, um die Ausgabe des Sprachmodells zu beeinflussen. 
>   - **Direktheit**: Wie direkt geben Sie dem Sprachmodell Anweisungen?
>   - **Spezifität**: Wie detailliert beschreiben Sie die gewünschte Ausgabe?
>   - **Tonalität**: Welchen Stil oder welche Stimmung soll der generierte Text haben?

>[!note]- Material Prompt-Dimensionen im Einsatz
> >[!info] Allgemein
> > **ChatGPT** ist ein leistungsstarkes Sprachmodell, das auf verschiedene Arten von Prompts reagieren kann. Durch die Variation der **Prompt-Dimensionen** können Sie die Ausgabe von ChatGPT gezielt steuern und an Ihre Bedürfnisse anpassen.
> >
> > In dieser Übung werden Sie die **Dimensionen Direktheit, Spezifität und Tonalität** genauer untersuchen. Sie werden lernen, wie man Prompts mit unterschiedlichen Dimensionen erstellt und wie sich diese auf die Ausgabe von ChatGPT auswirken.
> 
> >[!abstract] Themenliste
> >- Die Vorteile von erneuerbaren Energien
> >- Die Zukunft der Arbeit
> >- Künstliche Intelligenz im Alltag
> >- Die Bedeutung von Datenschutz
> >- Die Herausforderungen des Klimawandels
> 
> >[!abstract] Anleitung zur Prompt-Erstellung
> >**Direktheit:**
> >- **Direkter Prompt:** "Schreibe einen Blogartikel über die Vorteile von Solarenergie."
> >- **Indirekter Prompt:** "Viele Menschen interessieren sich für die Vorteile von Solarenergie. Könntest du darüber einen informativen Blogartikel schreiben?"
> >
> >**Spezifität:**
> >- **Spezifischer Prompt:** "Schreibe einen Blogartikel über die Vorteile von Solarenergie. Der Artikel soll mindestens 500 Wörter lang sein und die folgenden Aspekte behandeln: Umweltfreundlichkeit, Kosteneinsparungen, Unabhängigkeit von fossilen Brennstoffen."
> >- **Unspezifischer Prompt:** "Schreibe einen interessanten Blogartikel über Solarenergie."
> >
> >**Tonalität:**
> >- **Sachliche Tonalität:** "Schreibe einen informativen Blogartikel über die Vorteile von Solarenergie."
> >- **Humorvolle Tonalität:** "Schreibe einen lustigen Blogartikel über die Vorteile von Solarenergie. Der Artikel soll die Leser zum Schmunzeln bringen und gleichzeitig über die Vorteile von Solarenergie informieren."
