#  "Erstellen Sie Ihren eigenen Persona Prompt"
### Auftrag Basic Level A 
####  LLMs im Gespräch mit Persönlichkeiten

> [!abstract] LLMs im Gespräch mit Persönlichkeiten
> In diesem Auftrag lernen Sie, den Persona Pattern anzuwenden, indem Sie einen einfachen Prompt erstellen, der ein LLM anweist, aus der Perspektive einer bekannten Persönlichkeit zu antworten. Sie wählen eine Persona aus einer vorgegebenen Liste und formulieren Fragen, die zu dieser Persona passen. Anschliessend testen Sie ihren Prompt mit einem LLM und analysieren die Ergebnisse.

>[!todo]- Ablauf
>- Wählen Sie eine **Persona** aus der Liste.
>- Recherchieren Sie Ihre gewählte **Persona**, um mehr über ihren Hintergrund, ihre Persönlichkeit und ihre Ansichten zu erfahren.
>- Formulieren Sie einen **Prompt**, der das LLM anweist, als die gewählte **Persona** zu antworten.
>- Testen Sie Ihren **Prompt** mit einem **LLM** (z.B. ChatGPT).
>- Analysieren Sie die Antworten des **LLMs** und bewerten Sie, ob die **Persona** glaubwürdig dargestellt wurde.

>[!quote]- Prompting
> Beim **Persona Pattern** geben Sie dem Sprachmodell eine **spezifische Rolle** vor, indem Sie im Prompt formulieren: "Antworte als [Persona]...".  Stellen Sie anschliessend Ihre Fragen so, als würden Sie direkt mit der **Persona** sprechen.

>[!note]- Material LLMs im Gespräch mit Persönlichkeiten
> >[!info] Allgemein
> >Der **Persona Pattern**  ist eine Technik im **Prompt Engineering**, die es erlaubt, **LLMs** (Large Language Models) anzuweisen, aus der **Perspektive einer bestimmten Persona** zu antworten. 
> >
> >Durch die **Festlegung einer Persona** können wir **spezifischere** und **nuanciertere Antworten** von **LLMs** erhalten. 
> >
> >[!quote]- Definition Persona
> >Eine **Persona** ist eine **fiktive Person**, die **repräsentativ** für eine **bestimmte Zielgruppe** oder einen **Benutzertyp** ist. Sie wird häufig im **Marketing** und im **Design** verwendet, um die **Bedürfnisse** und **Verhaltensweisen** von **Nutzern** besser zu verstehen. Im Kontext von **LLMs** kann eine **Persona** eine **reale Person** (z.B. Albert Einstein, Greta Thunberg) oder eine **fiktive Figur** (z.B. Harry Potter, Sherlock Holmes) sein.
